## üìä How to Convert HTML to PDF/PPT

### Method 1: Browser Print to PDF (Easiest)

1. **Open the HTML file:**
   - `Documentation/QBR_DECK.html`
   - `Documentation/ARCHITECTURE_ONE_PAGE.html`

2. **Open in browser:**
   - Right-click ‚Üí "Open with" ‚Üí Chrome/Firefox/Edge

3. **Print to PDF:**
   - Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
   - Select "Save as PDF" as destination
   - Click "Save"

4. **For PPT:**
   - Open PDF in PowerPoint
   - File ‚Üí Import ‚Üí PDF
   - Each page becomes a slide

### Method 2: Online Converter

1. **Go to:** https://www.ilovepdf.com/html-to-pdf
2. **Upload:** HTML file
3. **Convert:** Click "Convert to PDF"
4. **Download:** PDF file

### Method 3: Command Line (Advanced)

```powershell
# Install wkhtmltopdf first
# Then convert:
wkhtmltopdf Documentation/QBR_DECK.html QBR_DECK.pdf
```
